# Database-Project-Xavier-2026

Enclosed in these files is the database that tracks the sales, stock, and members of the Muskie Co stores.

To set up the script:
Download the files from the zipped folder. Once you have done that go to lines 8 , 36, 37, and 38.
Replace the fields there with the name of your database, what your user is called in MySQL Workbench, your password, and the host address.

If you filled out these criteria correctly upon running the program, you should see a screen like this:


What task would you like to perform?
1. Information Processing
2. Maintaining Merchandise and Inventory Records
3. Maintaining Billing and Transaction Records
4. Reports

This means you have successfully input the correct credentials.

To add, delete, or query the database:

Simply follow the options on the menu to get the desired operation. After that add the required information to get the desired result. As long as the data entered is valid, it should update or query the database accordingly.
